# Support

## Where to ask for help

- **Usage questions / setup help:** GitHub Discussions
- **Bug reports:** GitHub Issues (use the bug template)
- **Feature requests:** GitHub Issues (use the feature template)
- **Security concerns:** See `SECURITY.md`

## Before opening a bug

1. Check the troubleshooting wiki
2. Update to the latest ChoreOps version
3. If it is a dashboard issue, update dashboard dependencies such as button-card to the latest version
4. Enable debug logging and capture relevant logs

Troubleshooting wiki:
https://github.com/ccpk1/choreops/wiki/Technical:-Troubleshooting
